# GitHub setup tips for this project

## Repository name

Best option:

```text
exchange-life-dumb-ways-to-fail
```

## Description

```text
A browser-based microgame project about funny exchange-student challenges, built with HTML, CSS, and JavaScript.
```

## Topics

Use these GitHub topics:

```text
javascript
html
css
web-game
microgame
frontend
game-development
```

## First commit

```bash
git init
git add .
git commit -m "Initial frontend prototype"
git branch -M main
git remote add origin https://github.com/YOUR-USER/exchange-life-dumb-ways-to-fail.git
git push -u origin main
```

## Good commits after that

```bash
git commit -m "Add password minigame"
git commit -m "Add character creator"
git commit -m "Add local leaderboard"
git commit -m "Add fullscreen result videos"
git commit -m "Translate UI to English"
```

## What to write in the project README

Include:

- project idea;
- technologies used;
- current features;
- how to run locally;
- how the game loop works;
- screenshots or GIFs;
- future features;
- credits/team members.
